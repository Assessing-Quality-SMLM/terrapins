# smlm-frc
Tools for working with [FRC (Fourier Ring Correlation)] calculation[^1](https://doi.org/10.1038/nmeth.2448) [^2](10.1016/j.jsb.2013.05.004) specifically targetting SMLM applications. 

[^1]: Nieuwenhuizen, R.P.J. et al. Measuring image resolution in optical nanoscopy. Nature Methods 10, pages 557–562 (2013). https://doi.org/10.1038/nmeth.2448

[^2]: Banterle, N. et al. Fourier ring correlation as a resolution criterion for super-resolution microscopy. J Struct Biol. 2013 Sep; 183(3):363-367. 10.1016/j.jsb.2013.05.004
